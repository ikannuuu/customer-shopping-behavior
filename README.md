# Customer Shopping Behavior Analysis

## 📌 Project Overview
Analysis of 3900 customers' shopping behavior using Python and Power BI.

## 🛠️ Tools Used
- Python (Pandas, JupyterLab)
- PostgreSQL
- Power BI

## 📊 Key Insights
- Clothing is the top revenue category
- Young Adults are the highest spenders
- Subscribers give better ratings (3.76 vs 3.69)

## 📁 Files
- `customer_shopping_analysis.ipynb` - Jupyter Notebook
- `customer_shopping_behavior.csv` - Dataset
